## Code Blocks

Here's a simple JavaScript function:

```js
function fibonacci(n) {
  if (n <= 1) return n
  return fibonacci(n - 1) + fibonacci(n - 2)
}

console.log(fibonacci(10)) // 55
```

And some TypeScript with types:

```ts
interface User {
  name: string

}

const greet = (user: User): string => {

}
```
